LeetCode 刷题（一）

May 8, 2016 | Hits

Q2 的 OKR 里有一条是刷 20 道 LeetCode 算法题，Q2 都快过去一半了才开始动手，惭愧惭愧。从最简单的开始刷起~

Reverse String（简单）

题目:

Given s = “hello”, return “olleh”.

解题

看上去很简单一道题目，不过做的时候没有想到系统 API，所以开始遍历字符串，结果悲剧 Time Limit Exceeded 了。后来搜了一下发现一句话就可以翻转字符串，自己简直蠢。

Add Digits（简单）

题目:

Given num = 38, the process is like: 3 + 8 = 11, 1 + 1 = 2. Since 2 has only one digit, return it.

解题

用了一种最粗暴最土鳖的一种解决方法，提示中提到了Digital root，不过你懂得。

Maximum Depth of Binary Tree（简单）

题目:

Given a binary tree, find its maximum depth.

The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

解题

分别对左右树进行遍历求深度

Minimum Depth of Binary Tree（简单）

题目:

Given a binary tree, find its minimum depth.

The minimum depth is the number of nodes along the shortest path from the root node down to the nearest leaf node.

解题

分别对左右树进行遍历求深度。因为是判断小的情况，所以当左右树中有一个为 nil 时会返回 0，我们应该增加对这种情况进行判断。

Move Zeroes（简单）

Given an array nums, write a function to move all 0’s to the end of it while maintaining the relative order of the non-zero elements.

For example, given nums = [0, 1, 0, 3, 12], after calling your function, nums should be [1, 3, 12, 0, 0].

这题也用了一个比较粗暴的方法，直接将所有非零的数字放到前面去，然后将后面剩下的位子都填为 0。
