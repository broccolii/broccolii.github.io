Vimium 的快捷键

Jun 1, 2015 | Hits

Vimium 是一款能在 Chrome 里运用 Vim 快捷键的扩展。(键盘党必备
这里是一张 Vimium 的快捷键列表：

Navigating the page

Command Note
j 向下滚动一点
k 向上滚动一点
gg 到页面最上部
G 到页面最底部
d 向下翻一屏
u 向上翻一屏
r 重载页面
gs 查看页面源代码
yy 复制当前 URL 到剪贴板
p 在当前标签页打开剪贴板中的 URL
P 在新标签页打开剪贴板中的 URL
gu 跳转到当前 URL 的上一层
gU 跳转到当前 URL 的最高层
f 在当前标签页中打开一个链接
F 在新标签页中打开一个链接

Using the vomnibar

Command Note
o 在当前标签页中打开 URl、书签或历史记录
O 在新标签页中打开 URl、书签或历史记录
T 搜索已打开的标签页
b 在当前标签页打开书签
B 在新窗口中打开书签

Navigating history

Command Note
H 根据浏览历史后退一页
L 根据浏览历史前进一页

Manipulating tabs

Command Note
K, gt 向右移动一个标签页
J, gT 向左移动一个标签页
g0 移动到第一个标签页
g$ 移动到最后一个标签页
t 创建一个新标签页
yt 复制当前标签页
x 关闭当前标签页
X 恢复已关闭的标签页
W 将标签页移动到新窗口
<< 向左移动标签页
>> 向右移动标签页
